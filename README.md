# XLX299 Dashboard

The standard XLX dashboard modified:

* relies on [xlx-activity-go](https://github.com/DigitalVoiceNZ/xlx-activity-go), a [PocketBase](https://pocketbase.io/) app that monitors the XLX journal to show real-time activity on the main page
* restyled
* additional pages (information and thanks)
* polls Peanut API to monitor users
* displays message of the day (can be dismissed)
